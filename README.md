# Portfolio Website - Rinzin Dorji

This is a personal portfolio website showcasing work in Organic Agriculture, Sustainable Innovation, and Research.

## Structure
- `index.html`: Main entry point.
- `css/styles.css`: Custom styling.
- `js/script.js`: Interactive elements and logic.
- `js/tailwind.config.js`: Tailwind CSS configuration.

## Deployment

### Option 1: Surge.sh (Quickest)
Since you have Node.js installed, you can deploy instantly from the terminal:
1. Open the terminal.
2. Run: `npx surge`
3. Follow the prompts to create an account and pick a domain.

### Option 2: Netlify Drop
1. Go to [Netlify Drop](https://app.netlify.com/drop).
2. Drag and drop the `portfolio-website` folder onto the page.

### Option 3: GitHub Pages (Recommended for long-term)
1. Install Git from [git-scm.com](https://git-scm.com/).
2. Initialize a repository: `git init`
3. Commit files: `git add .` and `git commit -m "Initial commit"`
4. Create a repository on GitHub.
5. Push your code and enable GitHub Pages in the repository settings.
